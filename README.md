# 🛡️ Insurance Policy Renewal & Claims Integration System

## 📌 Project Summary

Developed an end-to-end ETL solution using **Informatica Intelligent Cloud Services (IICS)** for processing insurance policy, claims, and renewal data. The project integrates multiple insurance datasets, validates records, processes renewal transactions, supports incremental loading, generates business summaries, and automates execution using **Taskflow**.

Implemented key ETL workflows including:

🔹 Policy and Customer Integration  
🔹 Claim Validation  
🔹 Policy Renewal Processing  
🔹 Incremental Data Loading  
🔹 Renewal Summary Generation  
🔹 Taskflow Automation  

This project demonstrates practical implementation of ETL concepts, transformations, data validation, aggregation, and workflow orchestration in a real-world insurance use case.


## 🛠️ Technologies Used

- ☁️ Informatica Intelligent Cloud Services (IICS)  
- 🗄️ Oracle Database  
- 💻 SQL  
- 🔄 ETL  
- ⚡ Taskflow
